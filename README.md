## Генетический алгоритм

![](images/convergence_BFGS)
рис. 4 (сходимость генетического алгоритма)

при моделировании системы тонких линз (Дрейф-Квадуполь-Д-К-Д-К-Д) мы просчитывали оптимизацию положения линзы в пространстве и ее силы несколькими методами:
1) поиск по сетке дрейфов
2) случайный поиск
3) генетический алгоритм(BFGS)
4) параллельный поиск
но при моделировании системы толстых линз той же схемы (Д-К-Д-К-Д-К-Д) появились новые параметры в виде длин квадруполей, из-за этого все методы кроме BFGS считали оптимальные параметры слишком долго, поэтому мы оставили расчет только при помощи этого метода.

Метод L-BFGS-B находит минимум функции, используя информацию о её направлении наискорейшего спуска и аппроксимируя её кривизну на основе данных с нескольких прошлых шагов, что позволяет экономить память. На каждой итерации он определяет оптимальное направление движения, делает шаг в этом направлении и корректирует его с учетом заданных граничных ограничений на переменные (например, "переменная не может быть меньше нуля"). Процесс повторяется до тех пор, пока функция не перестанет заметно уменьшаться или градиент не станет близок к нулю, что означает достижение минимума.
